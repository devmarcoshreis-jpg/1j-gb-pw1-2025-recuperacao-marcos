# Desafio - Jo-Ken-Po!:

### Jo-Ken-Po!.

O Jo-Ken-Po! é um jogo tradicional que combina estratégia e sorte, onde os jogadores escolhem entre pedra, papel e tesoura,neste projeto, você jogará contra o computador. O objetivo é simples: vencer o computador escolhendo a opção que derrota a escolha dele. A pedra vence a tesoura, a tesoura vence o papel e o papel vence a pedra. Prepare-se para testar sua sorte e estratégia neste divertido jogo de Jo-Ken-Po! Vença o computador e mostre suas habilidades!

## Funcionalidades
- Tela inicial, tela de cadastro e tela de jogo.
- Animação de onda feita com CSS para a logo do site.
- Opção para escolher seu sexo e ícone responsivo alternando entre ``Masculino`` , ``Feminino`` e``Não Informar``.
- Sistema de pontuação para acompanhar o desempenho do jogador usando
``sessionStorage``.
- Feedback visual para mostrar o resultado de cada rodada.
- Ranking dos melhores jogadores usando ``thead``,``table``,``tbody``,``th`` e tags referentes a tabela para organizar os dados.


## Tecnologias utilizadas

## Passo a passo para executar o projeto.
1. Clone o repositório para sua máquina local usando o comando:
   ```
   git clone 1j-gb-pw2-2026-recuperacao-marcos
   ```
2. Navegue até o diretório do projeto:
   ```
   cd 1j-gb-pw2-2026-recuperacao-marcos
   ```
3. Abra o arquivo `index.html` em seu navegador para visualizar o site.
4. Interaja com o botão ``JOGAR AGORA``.

5. Faça seu cadastro e identifique seu sexo(opcional) para começar a jogar, caso não informe, aparecerá um aviso dizendo que seu nome e idade são obrigatórios.

6. Apos o cadastro, escolha entre pedra, papel ou tesoura para jogar contra o computador e acompanhe sua pontuação, ou no topo da página, no cabeçalho ou logo abaixo, no ranking de jogadores.

7. Para iniciar jogar com outro nome, basta clicar no botão ``Sair do Jogo`` e será redirecionado para a tela de cadastro novamente.

8. Repita os passos 4 a 6 para continuar jogando e melhorando sua pontuação,ou competindo com outros jogadores que usarem o mesmo dispositivo, divirta-se!

## Experiências e aprendizados 

Com esse projeto, consegui entender como a manipulação do DOM funciona, como criar elementos dinamicamente usando JavaScript e como usar o ``sessionStorage`` para armazenar dados em uma sessão de navegador. Além disso, aprendi a criar uma interface de cadastro simples e a implementar um sistema de pontuação para acompanhar o desempenho do jogador. Também tive a oportunidade de praticar minhas habilidades de design usando CSS para criar uma animação de onda para a logo do site. é um projeto que me permitiu aplicar os conhecimentos adquiridos durante o curso, e que me instigou a procurar soluções criativas para os desafios que surgiram durante o desenvolvimento.
